# Изменения в этой версии

1. Убран ввод адреса API на экране входа и регистрации.
   Для разработки приложение автоматически использует:
   `http://127.0.0.1:8000/api/`

2. При создании заявки приложение больше не определяет геолокацию автоматически при открытии экрана.
   Разрешение на геолокацию запрашивается только после нажатия кнопки «Определить автоматически».

3. Если пользователь не даёт разрешение на геолокацию, заявка всё равно может быть создана через выбор точки на карте.

4. Координаты больше не показываются пользователю в интерфейсе создания заявки и списке заявок.
   Они сохраняются внутри приложения и отправляются на сервер.

5. Экран карты переделан: теперь он не зависит от OpenStreetMap/WebView/Leaflet и работает как встроенная интерактивная карта-прототип.
   Пользователь нажимает на карту, ставит маркер и подтверждает место.

6. Экран создания заявки визуально выровнен: отдельные карточки для категории, местоположения и фото.

## Как запускать в разработке

1. Запусти Laravel:
   `D:\xamp\php\php.exe artisan serve`

2. Подключи телефон по USB и сделай проброс порта:
   `& "D:\android\sdk\platform-tools\adb.exe" reverse tcp:8000 tcp:8000`

3. Запусти приложение на телефоне.

Адрес API вводить больше не нужно.
