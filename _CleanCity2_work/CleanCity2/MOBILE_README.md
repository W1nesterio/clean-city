# CleanCity Android — обновлённая рабочая версия

Что уже есть в мобильном приложении:

1. Вход через Laravel API.
2. Сохранение токена пользователя.
3. Режим жителя:
   - создание заявки;
   - автоматическое определение геолокации;
   - выбор фото из галереи;
   - предпросмотр выбранного фото;
   - просмотр своих заявок.
4. Режим исполнителя:
   - просмотр назначенных заявок;
   - принятие заявки;
   - перевод заявки в работу;
   - завершение заявки с загрузкой фото после выполнения.
5. Портретная ориентация экрана заблокирована.
6. Добавлены разрешения на интернет, чтение изображений и геолокацию.

## Как тестировать на реальном телефоне через USB

1. Запусти Laravel:

```powershell
cd D:\xamp\htdocs\clean-city-app
D:\xamp\php\php.exe artisan serve
```

2. Подключи телефон по USB и сделай проброс порта:

```powershell
& "D:\android\sdk\platform-tools\adb.exe" reverse tcp:8000 tcp:8000
```

3. В приложении в поле адреса API укажи:

```text
http://127.0.0.1:8000/api/
```

4. Данные для входа:

```text
resident@clean-city.local / password
worker@clean-city.local / password
admin@clean-city.local / password
```

## Важно

Для финального продукта поле адреса API нужно убрать и прописать постоянный адрес сервера в коде, например:

```text
https://your-domain.by/api/
```
