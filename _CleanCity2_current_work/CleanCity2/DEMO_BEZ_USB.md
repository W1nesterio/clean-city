# Запуск проекта на защите без USB

Мобильное приложение подключается к тому же Laravel API, что и админ-панель. USB нужен только для удобной разработки через `adb reverse`. На защите есть два нормальных варианта без USB.

## Вариант 1 — через Wi-Fi / точку доступа

1. Ноутбук и телефон должны быть в одной сети.
2. На ноутбуке запустить backend:

```powershell
cd D:\xamp\htdocs\clean-city-app
D:\xamp\php\php.exe artisan serve --host=0.0.0.0 --port=8000
```

3. Узнать IPv4 ноутбука командой:

```powershell
ipconfig
```

4. В браузере телефона проверить:

```text
http://IP_НОУТБУКА:8000/api/ping
```

5. В приложении в поле адреса API указать:

```text
http://IP_НОУТБУКА:8000/api/
```

Если `api/ping` не открывается, обычно мешает брандмауэр Windows или изоляция клиентов в сети.

## Вариант 2 — через публичный туннель

Для демонстрации можно поднять временный публичный адрес через ngrok или Cloudflare Tunnel. Тогда телефон подключается к интернет-ссылке, а не к локальному IP.

Пример итогового адреса:

```text
https://example-tunnel.ngrok-free.app/api/
```

Такой вариант удобен, если Wi-Fi в аудитории нестабильный или телефон не видит ноутбук в локальной сети.

## Как будет работать система на защите

1. На ноутбуке запускается Laravel backend и MySQL.
2. В браузере открывается админ-панель: `/admin/login`.
3. На телефоне открывается мобильное приложение.
4. Житель создаёт заявку с фото.
5. Заявка сразу появляется в админ-панели.
6. Администратор назначает исполнителя.
7. Исполнитель в мобильном приложении видит назначенную заявку и меняет статус.
