# Stage 13: профиль исполнителя без SMS

Что изменено в мобилке:
- Убран ввод телефона, должности и любых дополнительных данных.
- Убрано подтверждение телефона и экран кода.
- Профиль показывает аватарку, имя, организацию, личную статистику и выполненные работы.
- Аватарка остаётся управляемой: сделать фото, выбрать из галереи, удалить.
- Фото загружается сразу после выбора, отдельной кнопки сохранения больше нет.

После распаковки:
- Android Studio → Build → Clean Project
- Android Studio → Build → Rebuild Project

Чтобы удалить оставшийся класс SMS-кода, запусти из корня CleanCity2:

```powershell
.\REMOVE_SMS_MOBILE_FILES.bat
```
