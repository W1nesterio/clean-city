Stage24: переработка доступности.

Что исправлено:
- Вход и регистрация больше не используют внутренние настройки после выхода из аккаунта.
- На входе и регистрации есть отдельный переключатель "Доступность".
- Если включить "Доступность" на входе/регистрации, включаются крупный интерфейс, голос, звук и вибро для этих экранов.
- После успешного входа/регистрации режим переносится внутрь приложения.
- При выходе переключатель доступности на входе сбрасывается.
- Голосовой помощник говорит конкретные действия и поля, а не повторяет только название экрана.
- Добавлен голосовой ввод на входе и регистрации.
- Звук и вибро применяются глобальнее к кликам.

Установка:
Распаковать содержимое архива прямо в папку CleanCity2 с заменой.
Потом Android Studio -> Build -> Clean Project -> Rebuild Project.
