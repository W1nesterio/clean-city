Stage 11: доработка профиля исполнителя.

Что изменено:
- Профиль открывается через круглый аватар справа сверху на главной.
- На главной у исполнителя оставлена одна кнопка задач, дубль убран.
- Круг "ЧГ" на главной убран.
- В профиле аватар стоит рядом с именем.
- Нажатие на аватар открывает меню: камера, галерея, удалить фото.
- Фото отображается круглым.
- Поле "Дополнительный контакт" убрано.
- Поле ввода кода подтверждения телефона исправлено: текст больше не пересекается с подсказкой.

Установка:
1. Распаковать содержимое mobile_DROP_IN прямо в папку CleanCity2 с заменой.
2. Android Studio → Build → Clean Project → Rebuild Project.
