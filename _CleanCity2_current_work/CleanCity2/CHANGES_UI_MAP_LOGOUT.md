# Изменения в мобильном приложении

## Что изменено

1. **Выход из аккаунта**
   - Убрана большая кнопка выхода внизу главного экрана.
   - Выход теперь находится в верхнем блоке справа как небольшая аккуратная кнопка.

2. **Выбор точки на карте**
   - На экране создания заявки добавлена кнопка `Указать точку на карте`.
   - Пользователь может выбрать место загрязнения вручную, если текущее местоположение определилось неверно.
   - Карта открывается отдельным экраном.
   - Точку можно выбрать нажатием по карте или перетаскиванием маркера.

3. **Автоматическое местоположение сохранено**
   - Кнопка `Определить автоматически` осталась.
   - Пользователь может выбрать: использовать текущую геолокацию или отметить точку вручную.

4. **Выравнивание интерфейса**
   - Подправлены высоты кнопок.
   - Отключены заглавные буквы на кнопках MaterialButton.
   - Увеличены кнопки действий исполнителя, чтобы текст не прятался.

5. **Исправлена ошибка в TicketListActivity**
   - Убрано случайное дублирование `new TicketStatusRequest(...)`, которое могло ломать сборку.

## Важное замечание

Экран выбора точки использует OpenStreetMap через WebView. Для отображения карты телефону нужен интернет.
