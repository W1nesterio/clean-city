# Изменения главного экрана

- Убраны фразы вида «Здравствуйте, Житель».
- Убрано отображение email на главной странице.
- Убрано прямое отображение роли пользователя.
- Главный экран стал похож на обычное мобильное приложение: верхний брендовый блок, карточка быстрого действия и компактная карточка «Мои заявки».
- Для жителя показываются действия: создать заявку и открыть мои заявки.
- Для исполнителя показывается рабочий экран с переходом к назначенным задачам.
- Для администратора оставлена подсказка, что админ-панель работает через сайт.
