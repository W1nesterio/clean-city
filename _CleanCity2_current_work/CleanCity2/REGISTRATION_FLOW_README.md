# Регистрация в мобильном приложении

Вход остаётся один: email + пароль.

После входа приложение получает роль пользователя от backend и открывает нужный режим:

- `resident` — кабинет жителя;
- `worker` — кабинет исполнителя;
- `admin` — подсказка открыть веб-админку.

## Регистрация жителя

На экране регистрации выберите:

- `Я житель — обычная регистрация`.

Приложение отправляет запрос на:

```text
POST /api/register
```

## Регистрация сотрудника ЖКХ

На экране регистрации выберите:

- `Я сотрудник ЖКХ — регистрация по коду`.

Появится поле `Регистрационный код ЖКХ`.

Приложение отправляет запрос на:

```text
POST /api/register-worker
```

Код создаётся в веб-админке backend в разделе `/admin/worker-codes`.
