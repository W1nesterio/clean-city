Stage 10: профиль исполнителя.

Установка:
1. Распаковать содержимое архива прямо в папку CleanCity2 с заменой.
2. В Android Studio выполнить Clean Project и Rebuild Project.

Изменения:
- у исполнителя профиль открывается через круглый аватар справа сверху;
- фото профиля отображается круглым, не квадратным;
- экран профиля стал компактнее;
- при изменении телефона появляется одноразовый код подтверждения;
- в локальной демо-версии код показывается на экране, чтобы не подключать SMS-сервис.
