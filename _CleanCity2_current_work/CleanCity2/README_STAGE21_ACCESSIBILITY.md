# Stage 21 — доступность

Добавлено без изменений backend и БД:

- кнопка голосового ввода комментария в создании заявки;
- запуск заявки сразу с голосовым вводом с главной жителя;
- чекбокс "Нужна помощь" — добавляет к заявке короткую пометку для администратора;
- увеличены кнопки категорий;
- добавлены contentDescription для важных элементов, чтобы экранный диктор лучше их озвучивал.

Проверка:
1. Зайти как житель.
2. Нажать карточку "Голосом".
3. Надиктовать комментарий.
4. Отметить "Нужна помощь" при необходимости.
5. Отправить заявку как обычно.
